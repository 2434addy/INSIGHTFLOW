# InsightFlow — MVP Scope Definition

## MVP Objective

Deliver a functional product that allows a marketing agency to connect one or more ad platforms, automatically pull performance data, and generate an AI-written client report — all within 30 minutes of first signup.

## In Scope (MVP)

### Authentication & Onboarding
- [x] Email/password registration with email verification
- [x] Google OAuth sign-in
- [x] Guided onboarding wizard (connect first platform → select client → generate first report)
- [x] Workspace creation (one per agency)

### Platform Integrations (Phase 1)
- [x] **Meta Ads** — Campaign-level metrics (spend, impressions, clicks, conversions, ROAS)
- [x] **Google Ads** — Campaign-level metrics (spend, impressions, clicks, conversions, CPA)
- [ ] Google Analytics 4 — deferred to Phase 2
- [ ] Shopify — deferred to Phase 2

### Data Layer
- [x] OAuth 2.0 token management with encrypted storage
- [x] Daily automated data sync (configurable per connection)
- [x] Unified data model normalizing cross-platform metrics
- [x] 90-day historical backfill on connection
- [x] Basic data validation (null checks, range validation)

### AI Report Generation
- [x] Single report template: "Monthly Performance Summary"
- [x] AI-generated executive summary (2–3 paragraphs)
- [x] Period-over-period comparison (current vs. previous period)
- [x] Top 3 insights auto-detected from data
- [x] Top 3 recommendations generated by AI
- [x] PDF export

### Dashboard
- [x] Overview dashboard with aggregated KPIs per client
- [x] Client list with connection status indicators
- [x] Basic charting (spend, conversions, ROAS over time)
- [x] Date range selector

### Team Management
- [x] Invite team members via email
- [x] Two roles: Admin and Member
- [x] Basic activity log

## MVP Technical Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Backend | Python (FastAPI) | Async-first, strong AI/ML ecosystem |
| Frontend | Next.js 14 (App Router) | SSR, RSC, excellent DX |
| Database | PostgreSQL (Supabase or RDS) | Relational data, strong ecosystem |
| Cache | Redis | Session management, rate limiting, job queues |
| Queue | Redis + Celery (or BullMQ) | Async data sync and report generation |
| AI | Claude API (Anthropic) | Superior reasoning for narrative generation |
| Storage | S3-compatible (AWS S3 / R2) | PDF storage, asset management |
| Auth | NextAuth.js + custom JWT | Flexible, supports OAuth providers |
| Hosting | AWS (ECS/Fargate) or Vercel + Railway | Cost-effective, scalable |

| **Total** | **~12 weeks** |

## MVP Success Criteria

1. A new user can sign up, connect Meta Ads, and generate their first AI report in < 30 minutes
2. Generated reports are factually accurate (data matches source platform within 1% tolerance)
3. AI insights are relevant and actionable (validated by 5 beta agency users)
4. System handles 50 concurrent users without degradation
5. Zero critical security vulnerabilities in pre-launch audit
